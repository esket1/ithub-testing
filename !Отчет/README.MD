# Allure
Отчет от Allure не доступен единым файлом, он доступен лишь при запуске виртуального сервера за счет особенностей работы Allure на моей ОС. Изначально создать HTML-отчет в едином файле в моем случае попросту не предоставляется возможным.
Открыть его можно, введя в консоль следующее:
``` allure open allure-report ```

# Pytest-Cov
Отчет от Pytest-Cov доступен в едином HTML файле и находится в папке htmlcov, файл назван index.html.
