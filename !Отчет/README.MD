# Allure
Отчет от Allure доступен по следующей ссылке:
``` https://esket1.github.io/allure/ ```

Кроме того, отчет можно запустить и локально, открыв данный проект в консоли и введя в нее следующую команду:
``` allure open allure-report ```

# Pytest-Cov
Отчет от Pytest-Cov доступен в едином HTML файле и находится в папке htmlcov, файл назван index.html.
